# Changelog

## [1.1.0] - 2026-02

### Added
- QR code in admin header with participant URL and pre-filled room code
- Click-to-enlarge modal (320x320) for scanning from distance
- URL parameter support for auto-filling room code (`/participant.html?room=XXXX`)

## [1.0.0] - 2025-12

### Initial Release
- Simultaneous real-time voting system
- Multiple room support with pre-defined templates
- Admin and participant interfaces
- Optional timer with progress tracking
- Statistics and results visualization
